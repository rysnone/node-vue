# 将本地代码托管到码云中，使用源代码管理工具SVN或Git:
（建好的项目中）
 1. 新建一个 .gitignore 文件（项目忽略文件）
 - 文件内容：
 - node_modules
 - .idea
 - .vscode
 - .git
 2. 新建 README.md 项目描述文件
 - 这是一个xx的项目
 3. LICENSE （开源协议，可以从mui-master中直接复制过来）
 - 使用git命令执行一些操作，先安装Git
 4. 项目终端输入命令：git init，创建本地仓库
 5. 在码云中创建仓库，执行后续操作
